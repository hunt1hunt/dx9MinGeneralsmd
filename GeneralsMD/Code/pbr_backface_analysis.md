# PBR 载具背光面偏暗原因分析

## 背光面 PBR 光照构成

```hlsl
// 直接光照（只贡献给朝向太阳的面）
result += (diffuseColor + D * F0) * c1.xyz * NdotL    // 只有 NdotL>0 的面才有

// 环境光照（背光面的唯一来源）
result += diffuseColor * c10.xyz * ao

// 平添提亮（当前被注释掉）
// result += diffuseColor * 5.0
// result += diffuseColor * 10.0
```

背光面 = 仅受 c10（环境光）影响。背面 NdotL=0，太阳光贡献为零。

## 三种可能根因的区分

| 现象 | 可能原因 | 验证方法 |
|:----|:---------|:---------|
| 背光面很黑，但高光面正常 | 环境光 c10 太低 | 找完全背光的垂直面，亮度几乎为黑 → c10 不够 |
| 侧面（与太阳成90°）也偏黑 | roughness 太低（太光滑） | 侧面 NdotL≈0.5 应有较亮漫反射，但高光太窄 |
| 整个模型都偏暗 | 太阳光 c1 本身太弱 | 正面亮度也不够 → c1 亮度不足 |
| 金属载具黑但建筑正常 | metalness 太高 | diffuseColor = albedo*(1-metalness)，0.65→仅剩28% |

## 快速测试方案

测试A：AVHUMMER 临时改为 0.60,0.10 → 侧背面变亮则根因是metalness太高
测试B：c10 环境光翻倍 → 背面变亮则根因是环境光不足
测试C：坦克从 0.25,0.75 改为 0.40,0.40 → 侧面改善则根因是roughness太低+metalness太高

## 建议调整方向

| 模型 | 当前值 | 建议 |
|:----|:------|:-----|
| 坦克 | 0.25,0.75 | ↑ roughness 到 0.40, ↓ metalness 到 0.50 |
| 悍马 | 0.40,0.45 | 合理，可微调 metalness 到 0.30 |
| 建筑 | 0.75,0.08 | 保持不动 |
| 直升机 | 0.30,0.55 | ↑ roughness 到 0.40 |
