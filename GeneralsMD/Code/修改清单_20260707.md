## 1. PBR 排除系统 —— 5 个模型走固定管线

### 1.1 主修复：根源阻断

**文件：** `Libraries/Source/WWVegas/WW3D2/meshmdlio.cpp:1917-1979`
**原理：** 在 `MeshModelClass::post_process()` 中，PBR 参数推导前检查模型名（`ContainerName.` 前缀），匹配则跳过 `Set_Legacy_PBR()`，使 `Has_Legacy_PBR()` 返回 false，永不进入任何 PBR 渲染路径。
```cpp
{	bool skipPBR = false;
    const char *modelName = Get_Name();
    if (modelName) {
        const char *excludedPrefixes[] = {
            "0qsnwateryy1.", "bloombox_r.", "bloombox_rx.",
            "qingwaddskybox.", "qsnboxmorning.",
            "bloomboxa.", "bloomboxb.", "bloomboxc.",
        };
        for (int pi = 0; pi < 8; pi++) {
            if (_strnicmp(modelName, excludedPrefixes[pi], strlen(excludedPrefixes[pi])) == 0) {
                skipPBR = true; break;
            }
        }
    }
    if (!skipPBR) {
        // 原有 PBR 参数推导代码
    }
}
```
